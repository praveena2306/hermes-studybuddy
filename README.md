# 📚 Hermes StudyBuddy

Hermes StudyBuddy is a simple AI-powered student assistant built using Python and Streamlit.

## Features

- 📖 Study Plan Generator
- 💼 Interview Tips
- 📝 Notes Summarizer

## Technologies Used

- Python
- Streamlit

## Project Goal

This project was created for the Hermes Agent Challenge on DEV Community.
